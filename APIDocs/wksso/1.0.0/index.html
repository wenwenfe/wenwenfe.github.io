  • WKSSO

社区搜索部QQ用户PTLogin登录模块

用法：

1.使用WKSSO对象

引入包文件

<script src="${page.getStaticFilesPathPrefix()}/deploy/js/lib/sso/pc/bundle.js" babel="true" md5="false"></script>
<script src="${page.getStaticFilesPathPrefix()}/deploy/js/lib/sso/wap/bundle.js" babel="true" md5="false"></script>

API:

  • WKSSO.doLogin: 执行登陆， PC支持无刷新登录， Wap登录后通过URL记录操作
  • WKSSO.doLoginOut: 注销登录
  • WKSSO.checkLoginState: 检查登录态
  • WKSSO.insureLoginAction: 确保用户处于登录态，执行回调
  • WKSSO.appendCallbackList: 添加回调函数到登录后的回调栈（仅PC）

登录钩子类：

含有以下类名的DOM元素被点击后，如果没有其他事件阻止冒泡，自动执行登录/注销

  • 1）登录的类：#login，.login，#s_login，.s_login
  • 2）退出的类#logout， .logout， #s_logout，.sogou_logout

2.模块方法

引入登录模块，通过暴露的工厂方法获取实例对象

import {wapCreateLoginFactory} from "Login.Wap.js";
let login = wapCreateLoginFactory();

import {pcCreateLoginFactory} from "Login.PC.js";
let login = pcCreateLoginFactory();
